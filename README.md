# PanelVerse 🌌

> *Explore the infinite universe of graphic novels*

## 📚 项目简介

**PanelVerse** 是一个现代化的 Graphic Novel 社区分享平台，用户可以记录、分享和发现优秀的图像小说。

### 核心特性

- 📖 **个人书库管理** - 记录和管理您阅读过的 graphic novels
- 🌍 **社区发现** - 探索其他读者喜爱的作品
- 🏷️ **智能分类** - 灵活的分类和标签系统
- ⭐ **评分评论** - 分享您的阅读体验
- 👥 **社交互动** - 关注、点赞、讨论
- 🔥 **热门推荐** - 发现本周、本月最受欢迎的作品

## 🛠️ 技术栈

### 前端
- **Next.js 14** - React 框架，支持 SSR/SSG
- **Vanilla CSS** - 完全自定义的现代设计
- **Framer Motion** - 流畅的动画效果

### 后端 & 基础设施
- **Cloudflare Pages** - 全球 CDN 部署
- **Cloudflare D1** - 无服务器 SQLite 数据库
- **Cloudflare R2** - 对象存储（书籍封面）
- **Auth.js** - Google + Twitter/X OAuth 认证

### 外部 API
- **Google Books API** - 书籍元数据
- **Open Library API** - 备用数据源

## 🎨 设计理念

PanelVerse 采用现代、简洁的设计风格，focus on：

- **视觉优先** - 精美的书籍封面展示
- **流畅体验** - 丝滑的动画和过渡效果
- **社区驱动** - 鼓励用户互动和分享
- **响应式设计** - 完美适配各种设备
